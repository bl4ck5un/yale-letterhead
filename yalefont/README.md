Place the Yale 2024 typeface .otf files here before compiling:

  Yale2024-Roman.otf
  Yale2024-Bold.otf
  Yale2024-Italic.otf
  Yale2024-BoldItalic.otf

Obtain the fonts (Yale affiliation required) at:
https://yaleidentity.yale.edu/core-identity-elements/yale-typefaces

The font files themselves are licensed and excluded from this repository via .gitignore.
